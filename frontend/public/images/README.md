# Logo Image

Please place your logo image file here with the name `logo.png`.

## Supported formats:
- PNG (recommended)
- JPG/JPEG
- SVG

## Recommended specifications:
- Maximum width: 200px
- Maximum height: 80px
- Format: PNG with transparent background (if possible)
- File name: `logo.png`

If the logo image is not found or fails to load, the text "Vanatvam" will be displayed as a fallback.
